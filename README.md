# 全民運動科公文小助手

純靜態版公文操作手冊查詢網站，可部署於 GitHub Pages。

## 本版更新

- 保留原本藍白雲朵療癒風格。
- 保留承辦人員、登記桌人員的選單與搜尋建議。
- 新增「基層主管」選單與 Google 式搜尋建議。
- 新增「核判長官」選單與 Google 式搜尋建議。
- 查詢結果優先顯示小助手整理步驟。
- 手冊原文依據預設收合，需要時再展開查看。

## GitHub Pages 上傳檔案

請將以下檔案上傳至 repository 根目錄：

- `index.html`
- `style.css`
- `app.js`
- `search-data.json`
- `mascot.png`
- `README.md`
