# 全民運動科公文小助手

這是一個可部署於 GitHub Pages 的純靜態公文操作手冊查詢網站。

網站用途是協助使用者依照不同身分，快速查詢高雄第三代公文系統操作手冊中的相關指引，並以「小助手整理步驟 + 手冊原文依據」的方式呈現。

## 功能

- 依使用者身分查詢對應操作手冊
- 提供小助手整理版操作步驟
- 保留手冊原文依據
- 顯示來源手冊、章節名稱與 PDF 頁碼
- 支援繁體中文關鍵字搜尋
- 可部署於 GitHub Pages
- 不使用後端、不使用資料庫、不使用伺服器

## 身分選項

- 承辦人
- 基層主管
- 核判長官
- 登記桌人員
- 機關收文人員
- 機關發文人員
- 研考人員
- 機關系統管理者
- 電子交換人員
- 影像掃描人員
- 檔案管理人員（學校）
- 檔案管理人員（機關）

## 檔案說明

- `index.html`：網站主頁
- `style.css`：網站樣式
- `app.js`：查詢與顯示功能
- `search-data.json`：操作指引資料與手冊原文依據
- `README.md`：專案說明文件

## 資料內容

`search-data.json` 包含：

- 小助手精選操作指引
- 12 份 PDF 操作手冊之可抽取全文頁面
- 每筆資料皆保留身分、來源手冊、章節、頁碼與原文依據

## 重要說明

「小助手整理步驟」是為了讓使用者更快理解操作流程。

「手冊原文依據」保留原始手冊抽取文字，作為查核依據。

實際操作仍應以手冊原文及機關規定為準。
