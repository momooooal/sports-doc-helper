# 全民運動科公文小助手

## 功能

- 依使用者身分查詢對應操作手冊
- 僅搜尋所選身分的資料
- 顯示手冊原文，不摘要、不改寫
- 支援繁體中文關鍵字搜尋
- 可部署於 GitHub Pages

## 檔案說明

- `index.html`：網站主頁
- `style.css`：網站樣式
- `app.js`：查詢功能
- `search-data.json`：手冊原文資料

## 使用方式

1. 選擇身分
2. 輸入查詢關鍵字
3. 點選「查詢指引」
4. 查看對應手冊原文

## 注意事項

`search-data.json` 內的 `exact_text` 必須放置 PDF 手冊抽出的原文，網站只會顯示該欄位內容，不會自動改寫。
