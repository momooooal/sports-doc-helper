# 全民運動科公文小助手

本版本保留原本藍白、雲朵、圓角、療癒風格介面，並先針對「承辦人」查詢體驗優化。

## 本版重點

- 承辦人查詢結果維持精準顯示，避免一次列出過多 PDF 原文頁面。
- 新增「不確定要查什麼？」承辦人常用情境選單。
- 新增類 Google 的即時建議：輸入幾個關鍵字後，會出現可點選的相關操作。
- 小助手整理步驟優先顯示。
- 手冊原文依據預設收合，需要時可自行展開查看。
